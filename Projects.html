Convex-Hull Algorithm Visualizer

A program that visuallizes the Gift-Wrapping (Convex-Hull) Algorithm for the user, using JavaScript and the p5.js library. The program will generate a list of random points in a plane, then the program will computing the convex hull of those finite set of points and visualizer it for the user. The user will be able to see the program in actions.

Coronavirus Data Visualizer

This is a web app that displays the newest coronavirus news based the Worldometer API. This is built on Javascript with the NodeJS library on MongoDB. The user is able to choose their country/worldwide, then the web app will display a map with circles in different countries that is sized according to how many cases there are, a graph of coronavirus case trends, and many other data visualizing tools.
